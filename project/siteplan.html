Project Overview

Site Name: MicroLearn

Purpose: To provide a platform for bite-sized learning modules that allow busy users to master technical skills in 5 minutes or less.

Target Audience: Working professionals and students looking for quick skill refreshments in web development.

Design Principles (PARC)

  • Proximity: Related lessons will be grouped into categories.
  • Alignment: A strict grid system will be used for lesson cards.
  • Repetition: Consistent button styles and heading sizes across all pages.
  • Contrast: Dark blue backgrounds with bright white text for high readability.

Color Palette

Primary: #2c3e50
Secondary: #3498db
Accent: #e67e22

Typography

Headings: 'Montserrat', sans-serif

Body: 'Open Sans', sans-serif

Wireframes

The site will utilize a mobile-first approach. On mobile, the navigation will be a hamburger menu. On desktop, it will be a horizontal flexbox bar.
